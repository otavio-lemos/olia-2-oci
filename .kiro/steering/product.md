# Product — OCI Specialist LLM

## Propósito

Especialista em Oracle Cloud Infrastructure (OCI) fine-tuned para responder perguntas técnicas sobre serviços OCI, com foco em precisão, profundidade técnica e baixa alucinação.

## Proposta de Valor

- **Especialização**: Domínio profundo de 88 tópicos do OCI (compute, database, networking, security, etc.)
- **Assistência Técnica**: Geração de comandos OCI CLI, Terraform, troubleshooting
- **RAG Híbrido**: Busca semântica (FAISS) + lexical (BM25) para contextualização
- **Inferência Local**: Executável em Apple Silicon (M3 Pro) sem dependência de API externa

## Capacidades Core

1. **Respostas Técnicas**: Explicações aprofundadas sobre arquitetura e serviços OCI
2. **Geração de Código**: Comandos OCI CLI, Terraform, políticas IAM
3. **Troubleshooting**: Diagnóstico e resolução de problemas em compute, database, network, storage
4. **Planejamento**: Recomendações para migração, governança, security

## Público-Alvo

- Engenheiros que trabalham com OCI
- Profissionais de cloud computing
- Times de arquitetura e operações em Oracle Cloud
